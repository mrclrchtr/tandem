## Overview

Scratch live validation of the reloaded `supi-flow` tools after the task-envelope parsing fix.

## Validation scope

- create a flow ticket
- persist overview content
- add a headline-only task via `supi_flow_task`
- add a detailed task via `supi_flow_task`
- verify task listing and canonical task detail doc creation
- edit one task
- remove one task
- complete the remaining task
- close the ticket with archive evidence
